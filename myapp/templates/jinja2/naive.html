3) Naive implementation of templates with StreamingHttpResponse()


Following the official documentation, you can manually create a Template, a Context, and call Template.render() with StreamingHttpResponse().
Doing so, you get the same result as with HttpResponse(), the page hangs until everything is rendered, and gets displayed all at once. You just end up adding an extra layer of complexity for the same result.

The context contains a string : {{mystring}}

And also a short list, that could be rendered very quickly :

    {% for n in mylist %}
  • {{n}}
    • {% endfor %}

The next element in context is a request that will take much longer (approx 5 seconds)

    {% for element in mylongresponse %}
  • {{element}}
    • {% endfor %}
